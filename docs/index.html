Sugarboard — Installation

Navigateur requis : Google Chrome ou Microsoft Edge, sur ordinateur (Mac / Windows) ou téléphone Android.
Non compatible avec iPhone / iPad, Safari ni Firefox.
  1. Brancher le Sugarboard à l'ordinateur avec un câble USB (câble de données, pas seulement de charge).
  2. Cliquer sur « Installer » ci-dessous.
  3. Dans la fenêtre qui s'ouvre, sélectionner le port série (souvent usbserial ou USB Serial), puis Connecter.
  4. Sélectionner « Erase device / Effacer », puis Installer. Ne pas débrancher pendant l'opération (environ 1 à 2 minutes).
Navigateur non compatible — ouvrir cette page dans Chrome ou Edge. Accès au port série refusé.
Firmware Sugarboard v513 · installation locale, aucune donnée ne transite par un serveur.
