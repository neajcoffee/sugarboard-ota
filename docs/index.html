Sugarboard — Réinstaller le firmware

Cet outil réinstalle la dernière version du firmware (v513) directement depuis le navigateur, via le câble USB. À utiliser si l'appareil est bloqué (ERR LOGIN, écran figé) ou pour le remettre à neuf.

⚠️ Navigateur requis : Google Chrome ou Microsoft Edge, sur ordinateur (Mac / Windows) ou téléphone Android.
❌ Ne fonctionne pas sur iPhone / iPad ni sur Safari / Firefox.
  1. Branche le Sugarboard à l'ordinateur avec un câble USB (câble de données, pas seulement de charge).
  2. Clique sur « Installer / Flasher » ci-dessous.
  3. Dans la fenêtre qui s'ouvre, choisis le port série (souvent usbserial ou USB Serial), puis Connecter.
  4. Choisis « Erase device / Effacer » puis Installer. Ne débranche pas pendant l'opération (~1 à 2 min).
Ce navigateur n'est pas compatible — ouvre cette page dans Chrome ou Edge. Accès au port série refusé.
Après l'installation — vérifie l'écran :

Après le logo SUGARBOARD, il doit alterner en boucle « ←APP » (bleu) et « WIFI→ » (vert) : c'est l'écran de configuration, l'appareil est vierge et prêt à être appairé dans l'application.

❌ Si tu vois autre chose (chiffre figé, erreur) → relance l'installation.

Firmware Sugarboard v513 · installation locale, aucune donnée ne transite par un serveur.
